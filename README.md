# IoT Website

Welcome to the **IoT Website** repository! This project showcases a full-featured website for IoT (Internet of Things) applications.

## 🚀 Project Overview

This website serves as a platform to demonstrate IoT projects, company information, and team members. It is built with standard web technologies (HTML, CSS, JavaScript) and includes structured pages and images.

Key features include:
- About Us section highlighting the team: CEO, CFO, CTO.
- Responsive design for modern browsers.
- Structured project content for IoT applications.
- Clean and user-friendly interface.
